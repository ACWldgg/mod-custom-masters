# mod-custom-masters

Модуль для игрового сервера **AzerothCore (WotLK 3.3.5a)**, объединяющий кастомных NPC: **Glyph Master** и **Gem Master**.

## ✨ Особенности
* **Glyph Master** (Шаблон ID: `910001`) — управляющий символами.
* **Gem Master** (Шаблон ID: `910002`) — продавец/мастер самоцветов.
* **Авто-установка базы данных**: при первом запуске сервера ядро автоматически зальет шаблоны NPC и их точки спавна.
* **Локация**: NPC автоматически заселяются на **GM-остров** (карта 1, координаты: 16222, 16257, 13).

## 🚀 Установка на чистый сервер
1. Перейдите в папку `modules/` вашего сервера:
   ```bash
   cd /path-to-your-server/modules/
2. Склонируйте этот репозиторий:
git clone [https://github.com/ACWldgg/mod-custom-masters.git](https://github.com/ACWldgg/mod-custom-masters.git)
3. Пересоберите CMake и скомпилируйте ядро заново.
